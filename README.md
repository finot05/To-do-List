# 📝 Todo List App

A simple and responsive Todo List application built using **HTML**, **CSS**, and **JavaScript**.

## Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- Clean UI with Font Awesome icons
